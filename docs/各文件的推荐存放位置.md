本文件中描述了"工具"（gongju) 中提供的各种文件推荐的存放位置。

## bin 中的文件

bin/build.sh

bin/strip_os.sh

bin/install_os_run.sh

bin/release_info.sh

bin/pack_os.sh

bin/live_os.sh

以上文件建议以链接的方式存放在系统构建指定架构目录中。


## tools 中的文件

建议将整个 tools 目录建议以链接的方式整体创建在系统构建指定架构目录中。


## env 中的文件

env/arch.data

env/function.sh

以上文件建议以链接的方式存放在系统构建指定架构目录的 env 目录中。

env/data 目录建议整个以链接的方式存放在系统构建指定架构目录的 env 目录中。
